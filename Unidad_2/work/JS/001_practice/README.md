
# Ejercicio: Calculadora Básica con JavaScript, HTML y CSS

## Descripción

Este ejercicio muestra cómo construir una calculadora web sencilla utilizando HTML, CSS y JavaScript. La versión inicial dispone de las funciones de **suma** y **resta** entre dos números, presentando la estructura básica en tres archivos separados (`index.html`, `styles.css`, `script.js`). El objetivo educativo es que los estudiantes entiendan la integración de lenguajes para la creación de interfaces interactivas y practiquen la ampliación de funcionalidades en un proyecto real[web:19][web:27].

## Estructura de archivos

- `index.html`: define los campos, botones y caja resultado de la calculadora
- `styles.css`: agrega el diseño y la presentación visual
- `script.js`: contiene la lógica para las operaciones de suma y resta

## Objetivos de aprendizaje

- Manipular elementos del DOM con JavaScript
- Programar funciones que respondan a eventos en la página web
- Utilizar variables y operadores matemáticos en JS
- Aplicar estilos básicos con CSS

## Actividad requerida

1. **Completa la calculadora**:  
   Agrega las funciones de **multiplicación** y **división** en el archivo `script.js`, y añade los botones y lógica correspondiente en `index.html`.

2. **Validación de campos**:  
   Implementa mensajes para el usuario si ingresa valores vacíos, no numéricos o intenta dividir por cero.

3. **Mejora visual** *(opcional)*:  
   Personaliza el estilo en `styles.css` para que la calculadora sea más atractiva y accesible.

## Ejemplo de interfaz

# Problemas adicionales para practicar JavaScript 
Realizar un validador de formularios que compruebe si los campos “nombre” y “correo” tienen valores al presionar el botón de envío, mostrando advertencias si algún campo está vacío.

Programar una galería de imágenes que, al hacer clic en los botones de “siguiente” y “anterior”, muestre diferentes fotos sin recargar la página.

Desarrollar una página con una lista de tareas donde puedas agregar, eliminar y marcar como completadas las actividades usando JavaScript para manipular los elementos HTML.

Crear un botón que cambie el color de fondo de toda la página aleatoriamente cada vez que se pulse, utilizando JavaScript para modificar el estilo CSS3 en tiempo real.
